# IRC

[Wikipedia List of IRC Commands](https://en.wikipedia.org/wiki/List_of_Internet_Relay_Chat_commands)

List channels:

```
/list
```

```
/list *keyword*
```

Enter channel:

```
/join <channel>
```

Leave channel:

```
/part
```

Send private message in channel:

```
/notice <nick>
```

Start private chat to nick

```shell
/msg <nick>
```

Shows lag to a person:

```shell
/ping <nick>
```

```shell
/quit
```

```shell
/help
```
